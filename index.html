Struttura base documento

Tag Descrizione Esempio
<!DOCTYPE html> Dichiara il tipo di documento (HTML5). 
<!DOCTYPE html>
<html> Radice della pagina HTML. 
<html lang="it">
  ...
</html>
<head> Metadati, titolo, link a CSS ecc. 
<head>
  <meta charset="UTF-8">
  <title>Titolo</title>
</head>
<body> Contenuto visibile della pagina. 
<body>
  Contenuto...
</body>

Testo (titoli, paragrafi, enfasi)

Tag Descrizione Esempio
<h1>...</h1> ... <h6> 6 livelli di titolo (dal più importante al meno importante). 
<h1>Titolo principale</h1>
<h2>Sottotitolo</h2>
<p>...</p> Paragrafo di testo. 
<p>Questo è un paragrafo di testo.</p>
<br> A capo forzato all’interno di un paragrafo. 
Riga 1<br>
Riga 2
<strong>...</strong> Testo con forte enfasi (di solito grassetto). 
<p><strong>Attenzione:</strong> testo importante.</p>
<em>...</em> Enfasi sul testo (di solito corsivo). 
<p>Io <em>penso</em> che sia giusto.</p>
<blockquote> Citazione lunga (blocco). 
<blockquote>
  <p>Testo della citazione.</p>
</blockquote>
<q> Citazione breve in linea. 
<p>Lui disse: <q>ciao</q>.</p>
<abbr> Abbreviazioni o acronimi con significato. 
<p><abbr title="HyperText Markup Language">
HTML</abbr> è un linguaggio di markup.</p>

Liste

Tag Descrizione Esempio
<ol>, <li> Lista ordinata (numerata). 
<ol>
  <li>Primo passo</li>
  <li>Secondo passo</li>
</ol>
Attributo: type="1|a|A|i|I"
<ul>, <li> Lista non ordinata (punti). 
<ul>
  <li>Pane</li>
  <li>Latte</li>
</ul>
Attributo: type="disc|circle|square"
Liste nidificate Una lista dentro un elemento <li>. 
<ul>
  <li>Dolci
    <ul>
      <li>Croissant</li>
      <li>Torta</li>
    </ul>
  </li>
</ul>
<dl>, <dt>, <dd> Lista di definizioni (termine + descrizione). 
<dl>
  <dt>HTML</dt>
  <dd>Linguaggio per strutturare pagine web.</dd>
</dl>

Link

Tag / Attributo Descrizione Esempio
<a> + href Link ipertestuale. 
<a href="https://www.esempio.com">
Vai al sito</a>
URL assoluto Puntano a un altro sito. 
<a href="https://www.miosito.it/pagina.html">Link</a>
URL relativo Pagine dello stesso sito. 
<a href="contatti.html">Contatti</a>
target="_blank" Apre il link in una nuova scheda. 
<a href="pagina.html" target="_blank">
Apri in nuova scheda</a>
Link mail / telefono Richiamano client email o chiamata. 
<a href="mailto:utente@example.com">
Mandami una mail</a>

<a href="tel:3331234567">Chiama</a>
id + link interno Permette di saltare a un punto della stessa pagina. 
<h1 id="top">Inizio pagina</h1>
...
<a href="#top">Torna su</a>

Immagini

Tag Descrizione Esempio
<img> Inserisce un’immagine. 
<img src="img/foto.jpg"
     alt="Descrizione dell'immagine">
alt importante per accessibilità.
Dimensioni Attributi width e height in px o %. 
<img src="img/foto.jpg"
     alt="Foto"
     width="300px" height="200px">
<figure> + <figcaption> Raggruppa immagine e didascalia. 
<figure>
  <img src="img/foto.jpg" alt="Descrizione">
  <figcaption>Testo didascalia</figcaption>
</figure>

Tabelle

Tag Descrizione Esempio
<table> Inizia una tabella. 
<table>
  ...
</table>
<tr> Riga della tabella. 
<tr>
  <td>A</td>
  <td>B</td>
</tr>
<td> Cella di dati. 
<td>Valore</td>
<th> Cella di intestazione (titolo riga/colonna). 
<th scope="col">Prezzo</th>
colspan, rowspan Estende la cella su più colonne o righe. 
<td colspan="2">Unita due colonne</td>
<td rowspan="3">Unita tre righe</td>
<thead>, <tbody>, <tfoot> Sezioni di una tabella grande (intestazione, corpo, piede). 
<table>
  <thead>...</thead>
  <tbody>...</tbody>
  <tfoot>...</tfoot>
</table>

Contenitori e layout (HTML5)

Tag Uso principale Esempio
<div> Contenitore generico a blocco (solo per layout/stile). 
<div class="box">
  <p>Paragrafo dentro il div</p>
</div>
<span> Contenitore inline (per stile su una parte di testo). 
<p>Testo <span class="evidenzia">importante</span> qui.</p>
<header> Intestazione della pagina o di una sezione (logo, menu...). 
<header>
  <h1>Nome sito</h1>
  <nav>...</nav>
</header>
<footer> Piè di pagina (copyright, contatti...). 
<footer>
  &copy; 2025 Azienda
</footer>
<nav> Sezione di navigazione principale (menu con link). 
<nav>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="contatti.html">Contatti</a></li>
  </ul>
</nav>
<section> Sezione tematica della pagina (blocco con lo stesso tema). 
<section>
  <h2>Servizi</h2>
  <p>Descrizione servizi...</p>
</section>
<article> Contenuto indipendente e riutilizzabile (articolo, post, scheda...). 
<article>
  <h2>Titolo articolo</h2>
  <p>Testo...</p>
</article>

CSS base (selettori e proprietà principali)

Concetto Descrizione Esempio
Selettore di tipo Seleziona tutti gli elementi con quel tag. 
p {
  color: blue;
}
.classe Selettore di classe (attributo class). 
<p class="evidenzia">Testo</p>

.evidenzia {
  background-color: yellow;
}
#id Selettore di id (univoco nella pagina). 
<h1 id="titolo">Ciao</h1>

#titolo {
  text-align: center;
}
Stile inline CSS direttamente sul tag (da usare poco). 
<p style="color:red; font-size:20px;">
Testo rosso</p>
Proprietà utili Base per impaginare. 
color: red;              /* colore testo */
background-color: #eee; /* sfondo */
font-size: 16px;        /* grandezza testo */
margin: 10px;           /* spazio esterno */
padding: 10px;          /* spazio interno */
border: 1px solid #000; /* bordo */
text-align: center;     /* allineamento testo */
overflow-x: attributo; Se il contenuto è più largo della box, mostra una scrollbar orizzontale. 
pre {
overflow-x: auto;
}
Attributo: auto (quando serve) | scroll (sempre)
white-space: attributo; Se il contenuto è più largo della box, mostra una scrollbar orizzontale. 
pre {
overflow-x: auto;
}
Attributo: auto (quando serve) | scroll (sempre)
